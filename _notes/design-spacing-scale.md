# Design Spacing Scale

This document outlines the spacing scale used in the high-fidelity designs, converted from pixels to rem units (assuming 16px = 1rem base font size).

## Spacing Values

| Name | Pixel Value | Rem Value |
|------|-------------|-----------|
| spacing-none | 0px | 0rem |
| spacing-4xs | 2px | 0.125rem |
| spacing-3xs | 4px | 0.25rem |
| spacing-2xs | 8px | 0.5rem |
| spacing-xs | 12px | 0.75rem |
| spacing-s | 16px | 1rem |
| spacing-m | 24px | 1.5rem |
| spacing-lg | 32px | 2rem |
| spacing-xl | 48px | 3rem |
| spacing-2xl | 64px | 4rem |
| spacing-3xl | 96px | 6rem |
| spacing-4xl | 128px | 8rem |

## Usage Notes

- These values should be used consistently across all components and layouts
- The rem values are based on a 16px base font size
- Use these predefined spacing values instead of arbitrary pixel values
- Consider using CSS custom properties or design tokens to implement these values
