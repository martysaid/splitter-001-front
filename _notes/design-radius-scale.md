# Design Radius Scale

This document outlines the border radius scale used in the high-fidelity designs.

## Radius Values

| Name | Pixel Value |
|------|-------------|
| radius-sm | 8px |
| radius-m | 12px |
| radius-r | 16px |
| radius-l | 24px |

## Usage Notes

- These values should be used consistently across all components
- Border radius values are kept in pixels as they represent fixed visual properties
- Use these predefined radius values instead of arbitrary pixel values
- Consider using CSS custom properties or design tokens to implement these values
